JAVASCRIPT LOOPS

  1. For Loop:- When we know exactly how many times the loop is going to be executed then we will use for loop.
    Syntax:- 
                for (initialization; condition; increment/decrement)  
                {  
                 code to be executed  
                }
  2. While Loop:- When you don't know exactly how many times the loop is going to be executed then we will use while loop.
    Syntax:- 
                    while (condition)  
                {  
                 code to be executed  
                 increment/decrement(++/--)
                }
  3. Do-while Loop:- The do...while statement creates a loop that executes a specified statement until the test condition evaluates to false. The condition is evaluated after executing the statement, resulting in the specified statement executing at least once.
    Syntax:- 
                    do{  
                    code to be executed  
                    increment/decrement(++/--)
                }while (condition);
